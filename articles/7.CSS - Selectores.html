¿Que son los selectores en CSS?

Estos definen sobre qué elementos se aplicará un conjunto de reglas CSS.

Selectores

Selector de tipo: Selecciona todos los elementos que coinciden con el nombre del elemento especificado.

            input{
                color: blue;
            }

Selector de clase: Selecciona todos los elementos que tienen el atributo de class especificado.

            .className{
                color: yellow;
            }

Selector de ID: Selecciona un elemento basándose en el valor de su atributo id. Solo puede haber un elemento con un determinado ID dentro de un documento.

            #nombre{
                color: orange;
            }

Selector universal: Selecciona todos los elementos. Opcionalmente, puede estar restringido a un espacio de nombre específico o a todos los espacios de nombres.

            *{
                margin: 0;
                padding: 0;
            }

Selector de atributo: Selecciona elementos basándose en el valor de un determinado atributo.

            [data-text="descripcion"]{
                color: red;
            }

Selector descendiente: Se aplican estilos a elementos dentro de otros.

            .wrap div{
                width: 200px;
                height: 200px;
            }

Selector hijo: Se aplican estilos a elementos hijos directos.

            div > a{
                color: purple;
            }

Selector hermano adyacente: Se aplican estilos a elementos que siguen a otros.

            header + section{
                background: pink;
            }

Selector hermano general: Se aplican estilos a elementos al mismo nivel.

            header ~ section{
                background: orange;
            }
