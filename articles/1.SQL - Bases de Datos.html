¿Que son las Bases de Datos?

Una base de datos es un conjunto de datos pertenecientes a un mismo contexto y almacenados para su uso posterior.

¿Que es SQL?

Un tipo de lenguaje de programación que te permite manipular y descargar datos de una base de datos.
Tiene la capacidad de realizar cálculos avanzados.
Es utilizado en la mayoría de empresas que almacenan datos en una base de datos.
Ha sido y sigue siendo el lenguaje de programación más usado para bases de datos relacionales.

¿Qué tipos de bases de datos SQL existen?

Oracle: Oracle PLSQL es una extensión de SQL usada en Oracle.

MySQL: Esta considerada la base de datos de código abierto más popular del mundo.

Microsoft SQL Server: Es un tipo de SQL desarrollado por Microsoft y tiene algunas características especiales como que soporta procedimientos almacenados.

PostgreSQL: Es otro tipo de base de datos SQL de código abierto. No es tan popular como MySQL pero está ganando terreno.

Comandos básicos de SQL

CREATE DATABASE: Crear una nueva base de datos.

CREATE DATABASE nombre_de_baseDeDatos;

CREATE DATABASE mi_baseDeDatos;

USE: Hace referencia a nuestra base de datos, para comenzar a utilizarla.

USE mi_baseDeDatos;

DROP DATABASE: Elimina la base de datos.

DROP mi_baseDeDatos;

CREATE TABLE: Crear una nueva tabla.

CREATE TABLE nombre_de_tabla (nombre_del_campo TIPO_DE_DATO PRIMARY KEY, nombre TIPO_DE_DATO(longitud));

CREATE TABLE nueva_tabla ( id INT PRIMARY KEY, nombre VARCHAR(20), apellido VARCHAR(20), cedula VARCHAR(15));

ALTER TABLE: Puede usar un comando ALTER TABLE para agregar, modificar o quitar columnas o restricciones.

ALTER TABLE table_name predicado;

donde el predicado puede ser cualquiera de los siguientes:

ADD COLUMN nombre_column TIPO_DE_DATO(longitud);
Agrega una columna nueva DROP COLUMN nombre_column;
elimina una columna ALTER TABLE mi_tabla ADD COLUMN descripcion VARCHAR(100); ALTER TABLE mi_tabla DROP COLUMN descripcion;

DROP TABLE: Elimina la tabla.

DROP TABLE nombre_tabla;

WHERE: Permite seleccionar qué filtro aplicar a los datos.

SELECT nombre FROM mi_tabla WHERE id=2;

INSERT INTO: Insertar datos.

INSERT INTO nombre_tabla VALUES(value_1, value_2)
Ingresa/Agrega un nuevo registro en la tabla INSERT INTO usuarios(nombre, apellido, cedula) values("jose", "parra", "v12345678");

SELECT: Seleccionar los datos.

SELECT * FROM nombre_tabla
Selecciona todos los datos de una tabla SELECT * FROM nombre_tabla WHERE id=id_consultar
Selecciona todos los datos de una tabla que cumplan cierta condición.

UPDATE: Actualizar los datos.

UPDATA mi_tabla SET nombre=nuevo_nomrbe, apellido=nuevo_apellido WHERE nombre="alejo" AND apellido="pernia";
Actualiza todos los datos de una tabla que cumplan cierta condición.

DELETE: Borrar datos.

DELETE * FROM nombre_tabla
Elimina todos los datos de una tabla DELETE * FROM mi_tabla WHERE id=id_elimnar
Elimina los datos de una tabla que cumplan cierta condición.
