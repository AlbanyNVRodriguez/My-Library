¿Que es el local storage?

Es una API de almacenamiento web que nos permite almacenar datos de manera local en el navegador, sin necesidad de realizar alguna conexión a una base de datos.

¿Como se usa?

supongamos que tenemos un carrito de compras.

            let carrito = [];

Este carrito debe ir agregando productos cada que el usuario lo solicite.

            document.addEventListener("DOMCOntentLoaded", function(){
                  renderCarrito();
            });
            document.addEventListener("click", e=>{
                  carrito.push(e.target.name);
                  renderCarrito();
            });
            console.log(carrito);
            // ["camisa", "zapato", "gorro"]

Pero que pasa si el usuario actualiza la pagina.

            console.log(carrito);
            // [] - carrito vacio porque se actualizo la pagina.

ahora, para no perder esos datos ante una recarga de la pagina o el cierre del navegador, utilizaremos el local storage

            // localStorage.getItem("nombre") - para obtener valor
            // localStorage.setItem("nombre", value) - para declara su valor
            if(localStorage.getItem("arrayCarrito") !== null & localStorage.getItem("arrayCarrito") !== ""){
                  let carrito = localStorage.getItem("arrayCarrito").split(",");
            }else{
                  let carrito = [];
                  localStorage.setItem("arrayCarrito", carrito);
            }
            document.addEventListener("DOMContentLoaded", function(){
                  renderCarrito();
            });
            document.addEventListener("click", e=>{
                  carrito.push(e.target.name);
                  localStorage.setItem("arrayCarrito", carrito);
                  renderCarrito();

            });
